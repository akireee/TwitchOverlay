# TwitchOverlay

A custom Twitch IRC client built with Electron

## Requirements

 - Node 8 or above
 - npm
 - Electron 2.0.4
 - Windows (x64)

## Installing

**Production:**
Download ./TwitchOverlay-win32-x64.zip, extract the files, and run TwitchOverlay.exe

**Development:**
```sh
git clone https://github.com/devakira/TwitchOverlay
cd TwitchOverlay
npm install
electron .
```
